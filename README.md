# Stretch Film Advice | Digital Platform

Professional advisory platform for **Egbert Kort**, specializing in the strategic synergy between pallet wrapping machinery and multilayer stretch film.

## 🎯 Business Objectives
- **Credibility & Authority:** A premium, editorial design language inspired by McKinsey.com.
- **Efficiency:** Showcasing a 30-50% reduction in film usage through technical optimization.
- **European Reach:** Supporting distributors across 17 countries.

## 🛠 Technical Foundation
- **Framework:** Next.js 15 (App Router) & React 19
- **Styling:** Tailwind CSS v4 (Design tokens via CSS variables)
- **Animation:** Motion for React (Subtle scroll-reveals)
- **Typography:** Instrument Serif (Headlines) & Inter (Interface)
- **Contact:** Integrated Formspree endpoint

## 🏗 Directory Overview
- `/src/app`: Page architecture and routing.
- `/src/components/ui`: Core design primitives (Container, Section, Button).
- `/src/lib/content.ts`: Single source of truth for all site copy (i18n ready).
- `/src/lib/metadata.ts`: SEO and OpenGraph configurations.

## 🚀 Deployment
Hosted on **Vercel**. Every push to the `main` branch triggers an automated production deployment.
